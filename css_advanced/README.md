Homepage Project

Overview

This project is a custom-designed homepage created to teach people how to smile and be happy.

Features

    •	Responsive design
    •	Video tutorials 
    •	Free trial 

Technologies Used

    •	HTML5
    •	CSS3
    •	JavaScript 

Installation

Requirements

Make sure you have the following installed:

    •	Web browser (latest version recommended)

Steps to Run

    1.	Clone the repository:
git clone https://github.com/nasri9/homepage-project.git
    2.	Open the index.html file in a web browser:
cd homepage-project
open index.html
